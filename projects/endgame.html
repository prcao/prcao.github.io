About
Endgame is a 2D bullet hell inspired Java game I built in the summer for fun. Having no prior programming knowledge at the time, developing this game was a great learning experience that introduced me to the programming world.

While there is no actual storyline, Endgame contains all of the core mechanisms I wanted to put in, such as:
  • an inventory/equipment system
  • dynamic lighting
  • custom abilities
  • status effects
  • a variety of projectile behaviors
  • interactable objects
The project was written in Java using Java.awt for graphics. It is no longer being worked on due to how inefficient using Java.awt for graphics is.

Want a demo? Unfortunately, because this was written in Java, you'll have to run it locally. A release can be found here!
